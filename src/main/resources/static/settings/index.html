      데이터 백업
      현재 데이터를 백업 테이블에 스냅샷으로 저장 (단일 스냅샷, 덮어쓰기)
    

    

      
      
      
    

    

      분석데이터: -
       · 
      호출이력: -
    

    

  


  
  

    

      데이터 복구
      백업 스냅샷으로 현재 데이터를 덮어씁니다 — 되돌릴 수 없습니다.
    

    

      
      
      
    

    
복구 시 현재 데이터가 백업본으로 교체됩니다. 백업 없이 복구하면 오류가 발생합니다.

    

  


  


  
  

    

      데이터 삭제
      이 작업은 되돌릴 수 없습니다. 비밀번호 재확인 후 실행됩니다.
    

    

      
      
      
    

    

      분석데이터: URL 추출/분석 결과 (api_record)   ·  
      호출이력: APM 수집 데이터 (apm_call_data, MOCK/WHATAP/JENNIFER 모두)
    

    

  


  


  
  

    

      APM 호출이력 아카이브 (장기 보존)
      지정 기간 이전 데이터를 CSV로 백업 후 DB에서 제거합니다.
    

    

      
      
      
      
    

    

    

    

      파일 저장 위치: ./data/archive/apm_call_data_before_YYYY-MM-DD_archived_YYYY-MM-DD.csv
