URLViewer 업무 플로우 개요
URLViewer는 Spring Boot Controller 소스에서 URL을 자동 추출하고, APM 호출건수와 Git 커밋 이력을 결합해 차단 대상 여부를 판정하는 내부 도구입니다. 아래 플로우는 관리자가 소스를 추출·설정하고, 시스템 배치가 주기적으로 데이터를 수집하며, 상태 판정 로직이 자동 계산한 결과를 현업 검토자가 최종 확인하는 흐름을 보여줍니다.
관리자 자동 배치 상태 판정 사용자·검토

7. URL 상태 판정 플로우차트
